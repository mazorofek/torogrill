# Google Indexing Checklist

Use this after the production site is live on `https://toro-grill.com/`.

## 1. Connect the Domain

Make sure the final domain opens the live site, for example:

```text
https://toro-grill.com/
```

The site must be public, fast, and not blocked by authentication.

## 2. Add the Site to Google Search Console

1. Go to Google Search Console.
2. Add a new property for the final domain.
3. Verify ownership through DNS.
4. Use URL Inspection for the homepage.
5. Click Request Indexing.

## 3. Add a Google Business Profile

For restaurant searches, this is usually more important than organic SEO.

1. Create or claim the Google Business Profile for Toro Grill.
2. Use the same name, address, and phone:
   - Name: `טורו גריל`
   - Address: `סחרוב 20, ראשון לציון`
   - Phone: `03-952-0450`
3. Add the website URL: `https://toro-grill.com/`.
4. Add photos, opening hours, menu, and restaurant categories.

## 4. SEO Files Already Configured

These values are already configured for `https://toro-grill.com/`:

- Canonical URL
- Open Graph URL
- Absolute Open Graph image
- Absolute Twitter image
- `robots.txt` sitemap link
- `sitemap.xml`

## 5. Expected Timing

Google can take a few days to a few weeks to show a new site. Search Console indexing and a complete Google Business Profile usually speed this up.
